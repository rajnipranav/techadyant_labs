Orbitflow

A SaaS MVP for async engineering teams. A robust product engineering base powers a multi-agent layer that automatically summarizes updates, flags blockers, and nudges teammates.

Live in production
The Challenge Distributed teams struggle to extract the signal from the noise of daily async standups. Teams stop reading the updates completely, causing critical blocking dependencies to slip and projects to delay.
Implementation

Serverless product engineering meeting LangGraph AI.

  • Next.js + Supabase CoreBuilt the core SaaS experience using modern Next.js 14 and Supabase for real-time data push and role-based access control.
  • LangGraph Multi-agent workflowImplemented a complex state machine using LangGraph. Separate agents handle summarization, blocker-detection, and Slack nudging to prevent conflated reasoning.
  • Observability with LangSmithEvery multi-agent run is fully traced in LangSmith, ensuring prompt drift and token exhaustion can be detected immediately in production.
  • Stripe Billing StubTied all agent operational token consumption back to simple Stripe usage-based billing mechanisms for the overarching MVP.
